Twitter Sentiment analysis



Twitter sentiment analysis is a method of analyzing the emotions from tweets. These tweets are helpful in extracting the sentimental values from user. The proposed system is to analyze the sentiment of people using twitter API, Random Forest and predict the sentiment as positive, negative, neutral.
